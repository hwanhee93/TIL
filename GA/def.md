- Session
    + 세션은 방문자가 방문 시에 발생하는 하나의 지표이다.
    + 세션은 상호작용이 전송되는 순간 시작된다.
    + 세션은 다음과 같은 상황에 종료된다.
        * 30분동안 아무 이벤트가 발생하지 않는 경우
        * 24시(자정)이 된 경우
        * 홈페이지 유입 경로가 변경된 경우 
        
- 순 방문자수 vs 총 방문자수
    + 총 방문자수 = 세션 수
    + 순 방문자수 = UV (Unique Visitor)
    
- 페이지 수 (PV)
    + 페이지가 조회된 횟수
    
- 이탈율
    + 이탈수 : 첫 페이지만 보고 사이트를 나간 횟수
    + 이탈율 : 이탈수 / 총 방문수
